Recursos para desarrolladores web que están iniciando en este mundillo

< !DOCTYPE html >

La etiqueta doctype declara el tipo de documento, por lo que esta está señalando que este es un documento HTML.

< html ></ html >

Delimita el documento HTML.

< head></ head >

El elemento head delimita la cabecera del documento, entre sus etiquetas contiene información como scripts, metadatos, estilos, ubicación de documentos de estilos, título de la página, etc.

< h1 ></ h1 >

Etiqueta de encabezado de primer nivel. Se utiliza para titulos principales. Está etiqueta solo se puede utilizar una vez por docuemento HTML.

< h2 ></ h2 >

Etiqueta de encabezado de segundo nivel. Se utiliza para subtitulos de un h1. Está etiqueta solo se puede utilizar si antes hay un h1. Se puede utilizar varias veces en un mismo docuemnto HTML.

< h3 ></ h3 >

Etiqueta de encabezado de tercer nivel. Se utiliza para subtitulos de un h2. Se puede utilizar varias veces en un mismo docuemnto HTML.

< header ></ header >

En el header se coloca el contenido que se verá en la cabecera del sitio, por ejemplo la etriqueta h1 o un logo de la marca.

< main ></ main >

En esta etiqueta se coloca el contenido principal de la marca. Algunos desarrolladores la colocan entre el header y el footer (inicia cuando termina la etiqueta header y finaliza cuando inicia la etiqueta footer).

< footer ></ footer >

Se utiliza para colocar el pie de la página, es la utima etiqueta antes del cierre del body. Algunos de los elementos que se suelen colocar dentro de ella son: enlaces hacia las redes sociales, formulario de newsletter, mapa del sitio, etc.

< article ></ article >

Etiqueta de encabezado de tercer nivel. Se utiliza para subtitulos de un h2. Se puede utilizar varias veces en un mismo docuemnto HTML.
